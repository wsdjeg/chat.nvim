# Usage Guide

Learn how to use chat.nvim effectively.

---

## Core Concepts

chat.nvim provides:

- **Chat Interface**: Floating window for AI conversations
- **Tool System**: Execute commands, read files, search code, and more
- **Memory System**: Remember context across sessions
- **Multiple Providers**: Switch between AI providers seamlessly

## Basic Workflow

1. Open chat: `:Chat`
2. Ask questions or use tools: `@read_file ./src/main.lua`
3. Review AI responses
4. Apply suggestions to your code
