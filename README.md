# chat.nvim

A lightweight and extensible pure-Lua chat plugin for Neovim,

![chat.nvim](https://github.com/user-attachments/assets/bc51314c-a983-4b9e-9056-e1bf4dc51acc)

## Installation

Using nvim-plug:

```lua
require('plug').add({
  { 'wsdjeg/chat.nvim' },
})
```

## Usage

Use `:Chat` command to open this plugin.
